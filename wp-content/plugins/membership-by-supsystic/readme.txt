== Installation ==

1. Download Membership by Supsystic plugin
2. Unarchive (Extract) membership-by-supsystic.zip
3. Copy the folder with plugin
4. Open ftp \wp-content\plugins\
5. Paste the membership-by-supsystic plugin folder in the folder
6. Go to admin panel => open item "Plugins" => activate Membership by Supsystic

== Frequently Asked Questions ==

= How to start with Membership by Supsystic? =

First, you should install it. Then, to create your Membership you can follow detailed tutorial [Getting Started](https://supsystic.com/documentation/membership-getting-started?utm_source=wordpress&utm_medium=getting-started&utm_campaign=membership "Getting-started" and look [Membership Documentation](https://supsystic.com/docs/membership?utm_source=wordpress&utm_medium=documentation&utm_campaign=membership "WordPress Membership plugin documentation") to take full advantage of the plugin.

= How to limit access for specific content? =
In the Roles tab of Membership plugin, you can set an [Access to user profiles with specific roles parameter](https://supsystic.com/documentation/limit-access-specific-content?utm_source=wordpress&utm_medium=rolesparameter&utm_campaign=membership "Access to user profiles with specific roles parameter"). This allows you to expand or limit the access to the user with a specific role. 
Also, you can use a Membership Addon by Supsystic – [Restrict Content by User Roles](https://supsystic.com/extensions/mega/membership/membership-restrict-content/?utm_source=wordpress&utm_medium=restrictions&utm_campaign=extensions/mega/membership/membership-restrict-content "Restrict Content by User Roles") It gives a possibility to monetize content of your Membership website by limiting access to specific posts/pages by certain user roles.
To do this, follow the next steps:
1. [Install and activate](https://supsystic.com/extensions/mega/membership/membership-restrict-content/#faq "How to install PRO add-on of Membership plugin?") extension.
2. Go to the post/page you need and click "Edit". 
3. On the right sidebar of post/page editor find Supystic Membership Role section. 
4. Select membership roles that will have access to this post/page. 

= How to create dripping content? = 
To make a posting schedule and keep users to coming back for content, you need to follow this short instruction:
Firstly, install [Membership E-Commerce addon](https://supsystic.com/extensions/mega/membership/membership-ecommerce/?utm_source=wordpress&utm_medium=ecommerce&utm_campaign=extensions/mega/membership/membership-ecommerce "Membership E-commerce"). After installing and activating, go to any Pages or Posts and adjust the [settings](https://supsystic.com/documentation/dripping-content?utm_source=wordpress&utm_medium=drippingcontentsettings&utm_campaign=membership "Dripping Content") 
Secondly, navigate to E-commerce/Dripping Content where you can see all the posts and pages already available for dripping.

**[Photo Gallery plugin](https://supsystic.com/documentation/integration-contact-form-plugin?utm_source=wordpress&utm_medium=integration-with-gallery&utm_campaign=membership "Integration with Photo Gallery")** that provides an option to add amazing galleries with different templates, effects, video, captions and icons and more features to the membership community.
**[Slider plugin](https://supsystic.com/documentation/integration-contact-form-plugin?utm_source=wordpress&utm_medium=integration-with-slider&utm_campaign=membership " Integration with Slider plugin")** that allows posting slider to membership activities and your users will be able to post photos with your slider presets as well.

= Can I make posts in Membership Activity and share them via Facebook and Twitter account? 
You can do this and even more with [Membership Social Network Integration Addon](https://https://supsystic.com/extensions/mega/membership/membership-social-network-integration/?utm_source=wordpress&utm_medium=sociallogin&utm_campaign=extensions/mega/membership/membership-social-network-integration "Social Network Integration") It offers you next advantages:
* Import data from Social Networks to Membership Activity.
* Export data from Activity to Social Networks.
* Import/Export data by hashtags.
Follow detailed tutorial [Getting started](https://supsystic.com/documentation/social-network-integration?utm_source=wordpress&utm_medium=social-network-settings&utm_campaign=membership "Getting started") to take full advantage of the plugin.

= How to set front-end registration and login? 
After installation and activation Membership by Supsystic automatically will be created Membership Login and Registration pages. Simply add these pages to your site menu. Follow [this tutorial](https://supsystic.com/membership-registration-login?utm_source=wordpress&utm_medium=registrationlogin&utm_campaign=membership "Login/registration") for more information. 

Also with [Social Login addon](https://supsystic.com/extensions/mega/membership/membership-social-login/?utm_source=wordpress&utm_medium=sociallogin&utm_campaign=extensions/mega/membership/membership-social-login "Social Login add-on") you can enable Facebook and Twitter registration/login.  Your users have an ability to manage connections to personal Facebook and Twitter accounts at Social Sign-in section of Main Settings Tab in Membership Profile. Managing is simplified to 2 buttons “Connect” and “Disconnect”. 
Read more about [Social Login settings](https://supsystic.com/documentation/membership-social-login?utm_source=wordpress&utm_medium=social-login-settings&utm_campaign=membership "Social Login") here.

= How to integrate Membership and Twitter App? =
To connect the Membership plugin with Twitter you need to create your Twitter Application at first. In order to do that follow the simple steps from this [article](https://supsystic.com/integrate-membership-twitter?utm_source=wordpress&utm_medium=Twitterintegration&utm_campaign=membership "Twitter integration").

= How to create Auto Subscribe after new user Registration? =
With [Membership Subscriptions](https://supsystic.com/extensions/mega/membership/membership-subscriptions/?utm_source=wordpress&utm_medium=description&utm_campaign=extensions/mega/membership/membership-subscriptions "Membership Subscriptions") addon you have the ability to create subscriber directly after new user registration. Autosubscribe feature is available for MailChimp and Constant Contact engines. Look for more information about settings in [this tutorial](https://supsystic.com/documentation/membership-subscriptions?utm_source=wordpress&utm_medium=Subscriptionssettings&utm_campaign=membership "Subscriptions").
